04.йқўеҗ‘еҜ№иұЎ-дёҠ вҖӢ

дёҖ.зұ»дёҺеҜ№иұЎ вҖӢ

1.йқўеҗ‘еҜ№иұЎеӯҰд№ зҡ„дёүжқЎдё»зәҝ вҖӢ

  • Javaзұ»еҸҠзұ»зҡ„жҲҗе‘ҳпјҡеұһжҖ§гҖҒж–№жі•гҖҒжһ„йҖ еҷЁпјӣд»Јз Ғеқ—гҖҒеҶ…йғЁзұ»
  • йқўеҗ‘еҜ№иұЎзҡ„дёүеӨ§зү№еҫҒпјҡе°ҒиЈ…жҖ§гҖҒз»§жүҝжҖ§гҖҒеӨҡжҖҒжҖ§гҖҒпјҲжҠҪиұЎжҖ§пјү
  • е…¶д»–е…ій”®еӯ—пјҡthisгҖҒsuperгҖҒstaticгҖҒfinalгҖҒabstractгҖҒinterfaceгҖҒpackageгҖҒimportзӯү

2.йқўеҗ‘еҜ№иұЎдёҺйқўеҗ‘иҝҮзЁӢ(зҗҶи§Ј) вҖӢ

  • йқўеҗ‘иҝҮзЁӢпјҡејәи°ғзҡ„жҳҜеҠҹиғҪиЎҢдёәпјҢд»ҘеҮҪж•°дёәжңҖе°ҸеҚ•дҪҚпјҢиҖғиҷ‘жҖҺд№ҲеҒҡгҖӮ
  • йқўеҗ‘еҜ№иұЎпјҡејәи°ғе…·еӨҮдәҶеҠҹиғҪзҡ„еҜ№иұЎпјҢд»Ҙзұ»/еҜ№иұЎдёәжңҖе°ҸеҚ•дҪҚпјҢиҖғиҷ‘и°ҒжқҘеҒҡгҖӮ

3.йқўеҗ‘еҜ№иұЎдёӯдёӨдёӘйҮҚиҰҒзҡ„жҰӮеҝө вҖӢ

зұ»пјҡеҜ№дёҖзұ»дәӢзү©зҡ„жҸҸиҝ°пјҢжҳҜжҠҪиұЎзҡ„гҖҒжҰӮеҝөдёҠзҡ„е®ҡд№үгҖӮ

еҜ№иұЎпјҡжҳҜе®һйҷ…еӯҳеңЁзҡ„иҜҘзұ»дәӢзү©зҡ„жҜҸдёӘдёӘдҪ“пјҢеӣ жӯӨд№ҹз§°дёәе®һдҫӢ(instance)гҖӮ

  • йқўеҗ‘еҜ№иұЎзЁӢеәҸи®ҫи®Ўзҡ„йҮҚзӮ№жҳҜзұ»зҡ„и®ҫи®Ў
  • и®ҫи®Ўзұ»пјҢе°ұжҳҜи®ҫи®Ўзұ»зҡ„жҲҗе‘ҳ

дәҢиҖ…зҡ„е…ізі»пјҡеҜ№иұЎпјҢжҳҜз”ұзұ»newеҮәжқҘзҡ„пјҢжҙҫз”ҹеҮәжқҘзҡ„гҖӮ

4.йқўеҗ‘еҜ№иұЎжҖқжғіиҗҪең°е®һзҺ°зҡ„и§„еҲҷ вҖӢ

  1. еҲӣе»әзұ»пјҢи®ҫи®Ўзұ»зҡ„жҲҗе‘ҳ
  2. еҲӣе»әзұ»зҡ„еҜ№иұЎ
  3. йҖҡиҝҮвҖңеҜ№иұЎ.еұһжҖ§вҖңжҲ–вҖқеҜ№иұЎ.ж–№жі•вҖңи°ғз”ЁеҜ№иұЎзҡ„з»“жһ„

иЎҘе……пјҡеҮ дёӘжҰӮеҝөзҡ„дҪҝз”ЁиҜҙжҳҺ

  • еұһжҖ§ = жҲҗе‘ҳеҸҳйҮҸ = field = еҹҹгҖҒеӯ—ж®ө
  • ж–№жі• = жҲҗе‘ҳж–№жі• = еҮҪж•° = method
  • еҲӣе»әзұ»зҡ„еҜ№иұЎ = зұ»зҡ„е®һдҫӢеҢ– = е®һдҫӢеҢ–зұ»

6.JVM еҶ…еӯҳз»“жһ„ вҖӢ

зј–иҜ‘е®ҢжәҗзЁӢеәҸд»ҘеҗҺпјҢз”ҹжҲҗдёҖдёӘжҲ–еӨҡдёӘеӯ—иҠӮз Ғж–Үд»¶гҖӮ

жҲ‘д»¬дҪҝз”ЁJVMдёӯзҡ„зұ»зҡ„еҠ иҪҪеҷЁе’Ңи§ЈйҮҠеҷЁеҜ№з”ҹжҲҗзҡ„еӯ—иҠӮз Ғж–Үд»¶иҝӣиЎҢи§ЈйҮҠиҝҗиЎҢгҖӮж„Ҹе‘ізқҖпјҢйңҖиҰҒе°Ҷеӯ—иҠӮз Ғж–Үд»¶еҜ№еә”зҡ„зұ»еҠ иҪҪеҲ°еҶ…еӯҳдёӯпјҢж¶үеҸҠеҲ°еҶ…еӯҳи§ЈжһҗгҖӮ

иҷҡжӢҹжңәж ҲпјҢеҚідёәе№іж—¶жҸҗеҲ°зҡ„ж Ҳз»“жһ„гҖӮжҲ‘д»¬е°ҶеұҖйғЁеҸҳйҮҸеӮЁеӯҳеңЁж Ҳз»“жһ„дёӯ

е Ҷ,жҲ‘д»¬е°ҶnewеҮәжқҘзҡ„з»“жһ„пјҲжҜ”еҰӮпјҡж•°з»„гҖҒеҜ№иұЎпјүеҠ иҪҪеҲ°е Ҷз©әй—ҙдёӯгҖӮиЎҘе……пјҡеҜ№иұЎзҡ„еұһжҖ§пјҲйқһ static зҡ„пјүеҠ иҪҪеңЁе Ҷз©әй—ҙдёӯгҖӮ

ж–№жі•еҢәпјҡзұ»зҡ„еҠ иҪҪдҝЎжҒҜгҖҒеёёйҮҸжұ гҖҒйқҷжҖҒеҹҹ

дәҢ.зұ»зҡ„з»“жһ„д№ӢдёҖпјҡеұһжҖ§ вҖӢ

еҜ№жҜ”пјҡеұһжҖ§ vs еұҖйғЁеҸҳйҮҸ

1.зӣёеҗҢзӮ№ вҖӢ

  • е®ҡд№үеҸҳйҮҸзҡ„ж јејҸпјҡж•°жҚ®зұ»еһӢ еҸҳйҮҸеҗҚ = еҸҳйҮҸеҖј
  • е…ҲеЈ°жҳҺпјҢеҗҺдҪҝз”Ё
  • еҸҳйҮҸйғҪе…¶еҜ№еә”зҡ„дҪңз”Ёеҹҹ

2.дёҚеҗҢзӮ№ вҖӢ

  • еңЁзұ»дёӯеЈ°жҳҺзҡ„дҪҚзҪ®зҡ„дёҚеҗҢ
    • еұһжҖ§пјҡзӣҙжҺҘе®ҡд№үеңЁзұ»зҡ„дёҖеҜ№{}еҶ…
    • еұҖйғЁеҸҳйҮҸпјҡеЈ°жҳҺеңЁж–№жі•еҶ…гҖҒж–№жі•еҪўеҸӮгҖҒд»Јз Ғеқ—еҶ…гҖҒжһ„йҖ еҷЁеҪўеҸӮгҖҒжһ„йҖ еҷЁеҶ…йғЁзҡ„еҸҳйҮҸ
  • е…ідәҺжқғйҷҗдҝ®йҘ°з¬Ұзҡ„дёҚеҗҢ
    • еұһжҖ§пјҡеҸҜд»ҘеңЁеЈ°жҳҺеұһжҖ§ж—¶пјҢжҢҮжҳҺе…¶жқғйҷҗпјҢдҪҝз”Ёжқғйҷҗдҝ®йҘ°з¬ҰгҖӮ еёёз”Ёзҡ„жқғйҷҗдҝ®йҘ°з¬ҰпјҡprivateгҖҒpublicгҖҒзјәзңҒгҖҒprotected => е°ҒиЈ…жҖ§
    • еұҖйғЁеҸҳйҮҸпјҡдёҚеҸҜд»ҘдҪҝз”Ёжқғйҷҗдҝ®йҘ°з¬Ұ
  • й»ҳи®ӨеҲқе§ӢеҢ–еҖјзҡ„жғ…еҶө
    • еұһжҖ§пјҡзұ»зҡ„еұһжҖ§пјҢж №жҚ®е…¶зұ»еһӢпјҢйғҪй»ҳи®ӨеҲқе§ӢеҢ–еҖј ж•ҙеһӢ(byteгҖҒshortгҖҒintгҖҒlong : 0 ) жө®зӮ№еһӢ(float гҖҒ doubel : 0.0 ) еӯ—з¬ҰеһӢ(char : 0 (жҲ–'\u0000')) еёғе°”еһӢ(boolean : false) еј•з”Ёж•°жҚ®зұ»еһӢпјҲзұ»гҖҒж•°з»„гҖҒжҺҘеҸЈ : nullпјү
    • еұҖйғЁеҸҳйҮҸпјҡжІЎжңүй»ҳи®ӨеҲқе§ӢеҢ–еҖј ж„Ҹе‘ізқҖпјҢжҲ‘д»¬еңЁи°ғз”ЁеұҖйғЁеҸҳйҮҸд№ӢеүҚпјҢдёҖе®ҡиҰҒжҳҫзӨәиөӢеҖјгҖӮ зү№еҲ«ең°пјҡеҪўеҸӮеңЁи°ғз”Ёж—¶пјҢжҲ‘д»¬иөӢеҖјеҚіеҸҜгҖӮ
  • еңЁеҶ…еӯҳдёӯеҠ иҪҪзҡ„дҪҚзҪ®
    • еұһжҖ§пјҡеҠ иҪҪеҲ°е Ҷз©әй—ҙдёӯпјҲйқһ staticпјү
    • еұҖйғЁеҸҳйҮҸпјҡеҠ иҪҪеҲ°ж Ҳз©әй—ҙ

2.this и°ғз”ЁеұһжҖ§гҖҒж–№жі• вҖӢ

thisзҗҶи§ЈдёәпјҡеҪ“еүҚеҜ№иұЎ жҲ– еҪ“еүҚжӯЈеңЁеҲӣе»әзҡ„еҜ№иұЎ

  • еңЁзұ»зҡ„ж–№жі•дёӯпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёthis.еұһжҖ§жҲ–this.ж–№жі•зҡ„ж–№ејҸпјҢи°ғз”ЁеҪ“еүҚеҜ№иұЎеұһжҖ§жҲ–ж–№жі•гҖӮдҪҶжҳҜпјҢйҖҡеёёжғ…еҶөдёӢпјҢжҲ‘д»¬йғҪзңҒз•ҘthisгҖӮзү№ж®Ҡжғ…еҶөдёӢпјҢеҰӮжһңж–№жі•зҡ„еҪўеҸӮе’Ңзұ»зҡ„еұһжҖ§еҗҢеҗҚж—¶пјҢжҲ‘д»¬еҝ…йЎ»жҳҫзӨәзҡ„дҪҝз”Ёthis.еҸҳйҮҸзҡ„ж–№ејҸпјҢиЎЁжҳҺжӯӨеҸҳйҮҸжҳҜеұһжҖ§пјҢиҖҢйқһеҪўеҸӮгҖӮ
  • еңЁзұ»зҡ„жһ„йҖ еҷЁдёӯпјҢжҲ‘д»¬еҸҜд»ҘдҪҝз”Ёthis.еұһжҖ§жҲ–this.ж–№жі•зҡ„ж–№ејҸпјҢи°ғз”ЁеҪ“еүҚжӯЈеңЁеҲӣе»әзҡ„еҜ№иұЎеұһжҖ§жҲ–ж–№жі•гҖӮдҪҶжҳҜпјҢйҖҡеёёжғ…еҶөдёӢпјҢжҲ‘д»¬йғҪзңҒз•ҘthisгҖӮзү№ж®Ҡжғ…еҶөдёӢпјҢеҰӮжһңжһ„йҖ еҷЁзҡ„еҪўеҸӮе’Ңзұ»зҡ„еұһжҖ§еҗҢеҗҚж—¶пјҢжҲ‘д»¬еҝ…йЎ»жҳҫзӨәзҡ„дҪҝз”Ёthis.еҸҳйҮҸзҡ„ж–№ејҸпјҢиЎЁжҳҺжӯӨеҸҳйҮҸжҳҜеұһжҖ§пјҢиҖҢйқһеҪўеҸӮгҖӮ

3.this и°ғз”Ёжһ„йҖ еҷЁ вҖӢ

  2. жһ„йҖ еҷЁдёӯдёҚиғҪйҖҡиҝҮthisпјҲеҪўеҸӮеҲ—иЎЁпјүзҡ„ж–№ејҸи°ғз”ЁиҮӘе·ұгҖӮ
  3. еҰӮжһңдёҖдёӘзұ»дёӯжңүnдёӘжһ„йҖ еҷЁпјҢеҲҷжңҖеӨҡжңүn-1жһ„йҖ еҷЁдёӯдҪҝз”ЁдәҶthisпјҲеҪўеҸӮеҲ—иЎЁпјүгҖӮ
  4. и§„е®ҡпјҡthisпјҲеҪўеҸӮеҲ—иЎЁпјүеҝ…йЎ»еЈ°жҳҺеңЁеҪ“еүҚжһ„йҖ еҷЁйҰ–иЎҢгҖӮ
